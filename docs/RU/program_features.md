## 1) Окна приложения и компоновка

### 1.1 Окно приветствия
- **Список недавних проектов**.
- **Open** — открыть выбранный проект.
- **Open…** — выбрать папку проекта.
- **Create** — переход к странице создания проекта.

### 1.2 Создание проекта
- **Project Name** (валидатор: буквы, цифры, `_` и `-`).
- **Language**: C / C++ / ASM / C + ASM / Custom.
- **Path**: выбор родительской директории.
- **Create**: проверяет поля, создаёт папку и открывает IDE.
- **Back**: вернуться на окно приветствия.

### 1.3 Главное окно IDE
- **Меню** (File / Edit / View / Build / Tools / References).
- **Левая панель**: дерево файлов проекта.
- **Основная область**: вкладки файлов и вкладки инструментов.
- **Status bar**: стандартный статус‑бар Qt.
- **Панель терминала** (включается через View).

### 1.4 Сплиттеры и панели
- Горизонтальный сплиттер: дерево файлов ↔ рабочая область.
- Вертикальный сплиттер: рабочая область ↔ терминал.
- Основная область не сворачивается; терминал можно скрывать.

## 2) Дерево проекта / файлов

### 2.1 Поведение дерева
- Использует файловую систему ОС (read/write).
- Двойной клик по файлу открывает вкладку.
- Скрыты служебные колонки; заголовок скрыт.
- Видимость дерева переключается через **View → File Tree**.

### 2.2 Контекстное меню (ПКМ)
**На папке:**
- **Open** (раскрыть каталог).
- **Rename** (переименование inline).
- **Delete** (с подтверждением).
- **Create File** (диалог создания файла).
- **Create Folder** (диалог создания папки).

**На файле:**
- **Open**.
- **Rename**.
- **Delete** (с подтверждением).

**На пустой области:**
- **Create File** (в корне проекта).
- **Create Folder** (в корне проекта).

## 3) Вкладки файлов (верхняя панель)

### 3.1 Открытие и управление
- Каждый файл открывается в своей вкладке.
- Повторное открытие фокусирует уже открытый файл.
- **Индикатор изменений**: `*` в заголовке вкладки.

### 3.2 Закрепление вкладок
- **Pin/Unpin** через контекстное меню вкладки.
- Закреплённые вкладки переносятся в начало.
- Иконка **📌** (или тема `emblem-pinned`).
- **Закреплённые нельзя закрыть**, пока не откреплены.
- Ограничения перетаскивания:
  - закреплённые можно менять местами **только внутри зоны закреплённых**;
  - незакреплённые нельзя перетаскивать в зону закреплённых.

### 3.3 Закрытие вкладок
- Кнопка закрытия, **Ctrl+W**, либо **средняя кнопка мыши**.
- При наличии изменений — диалог **Save / Don’t Save / Cancel**.

### 3.4 Навигация по вкладкам
- **Alt + Left/Right** — переключение вкладок.
- **Alt + колесо мыши** — переключение вкладок.

## 4) Вкладки инструментов (для каждого файла)

Все инструменты разделяют общий буфер данных файла, изменения синхронизируются.

### 4.1 Code Editor
- **Подсветка и автодополнение** для:
  - C / C++ (`.c`, `.h`, `.cpp`, `.hpp`)
  - ASM (`.asm`, `.s`)
  - Rust (`.rs`)
  - Make (`Makefile`, `.mk`, `.make`)
  - CMake (`CMakeLists.txt`, `.cmake`)
- **Номера строк** + подсветка текущей строки.
- **Подсветка скобок**.
- **Авто‑отступы** и авто‑закрытие скобок.
- **Tab → 4 пробела**, Backtab — уменьшить отступ.
- **Zoom**: **Ctrl + Plus / Ctrl + Minus**.
- **Поиск**:
  - **Ctrl+F** открыть, **Esc** закрыть.
  - кнопки **Prev / Next** и авто‑поиск.
- **Binary file detected**: блок‑страница с кнопкой **Open anyway**.
- **Синхронизация выделения** с другими инструментами.

### 4.2 Binary (HEX / форматы)
Форматные страницы синхронизированы через общий буфер:

- **RAW**: hex‑редактор (QHexView).
  - Редактирование байтов (патчинг).
  - Синхронизация выделения.
  - Контекстное меню: **Select All / Copy / Paste**.
  - Стандартные шорткаты: Undo / Redo / Cut / Copy / Paste / Select All.
  - Диалог Find/Replace (Text/Hex/Int/Float, направления).

- **ELF**: проверка и показ заголовка.
  - Сигнатура ELF, Class, Endian, Type, Machine.

- **PE**: заголовок PE.
  - Проверка MZ + PE сигнатуры.
  - Machine, число секций, характеристики.

- **MBR**: Master Boot Record.
  - Проверка сигнатуры 0x55AA.
  - Таблица разделов: активность, тип, LBA, размер.

### 4.3 Disassembler
- **Backend**: objdump или radare2 (настраивается в Settings).
- **Toolbar**:
  - **Disassemble** / **Cancel**.
  - **Section** selector.
  - **Search** по адресу/мнемонике/операндам.
  - **Log** (панель диагностики).
- **Список функций** с фильтром и переходом.
- **Листинг**:
  - фиксированные колонки, auto‑comment для строк.
  - tooltip‑подсказки по мнемоникам (флаги, конвертация чисел).
- **Контекстное меню**:
  - Copy address / Copy bytes.
  - Hex patch here… (только radare2).
  - Patch bytes… (та же длина).
  - Hex patch string… (только radare2).
  - Show string bytes… (hex + ASCII).
- **F5** — повторный анализ.

## 5) Реверс‑калькулятор (Tools → Reverse Calculator)

- Ввод: десятичный, hex (`0x`), бинарный (`0b`), поддержка знака.
- Разрядность: 8 / 16 / 32 / 64.
- **Show signed**.
- **Swap endian**.
- Копирование: **Copy hex / dec / bin**.

---

## 6) Настройки (Edit → Settings)

### 6.1 Backend дизассемблера
- Выбор **objdump** или **radare2**.
- Пути к инструментам (авто‑поиск в PATH при пустом поле).
- Статусы: **found / missing** для objdump, r2 и `file`.

### 6.2 Параметры дизассемблера
- Лимит инструкций на секцию.
- Синтаксис ассемблера: **Intel / AT&T**.
- Анализ radare2: **None / aa / aaa**.
- Pre‑commands для r2 (по строкам).

### 6.3 Импорт / экспорт
- **Test**: проверка наличия инструментов.
- **Import… / Export…** настроек в INI.
