# Горячие клавиши

На macOS используйте `Cmd` вместо `Ctrl` для стандартных комбинаций!

| горячая клавиша | действие |
|---|---|
| Ctrl+S | Файл: Сохранить файл |
| Ctrl+, | Настройки |
| Ctrl+B | Файловое дерево: показать/скрыть |
| Ctrl+Shift+R | Калькулятор |
| Ctrl+Shift+D | Конвертация данных |
| Ctrl+Shift+S | Генерация Shellcode|
| F5 | Дизассемблер: перезапустить дизассемблирование |
| Ctrl+F | Редактор кода: открыть поиск |
| Esc | Редактор кода: закрыть поиск |
| Ctrl+Plus / Ctrl+= | Редактор кода: увеличить масштаб |
| Ctrl+Minus | Редактор кода: уменьшить масштаб |
| Ctrl+Scroll | Редактор кода: масштаб колесиком мыши |
| Ctrl+Space | Редактор кода: вызов автодополнения |
| ALT+MWheel UP/DOWN | Редактор кода: переключение между вкладками |
| ALT+Arrow LEFT/RIGHT | Редактор кода: переключение между вкладками |
| Ctrl+W | Редактор кода: закрыть текущую вкладку |
| Mouse Middle Button | Редактор кода: закрыть текущую вкладку |
| Tab | Редактор кода: вставить 4 пробела |
| Shift+Tab | Редактор кода: уменьшить отступ (до 4 пробелов) |
| Enter | Редактор кода: авто-отступ (включая `{}`) |
| Ctrl+` | Терминал: открыть / закрыть |
| Ctrl+C | Терминал: копировать выделение или прервать, если нет выделения |
| Ctrl+A | Терминал: выделить от последнего приглашения до конца |
| Enter | Терминал: выполнить команду |
| Up / Down | Терминал: история команд |
| Left / Right | HEX-редактор: переместить курсор |
| Shift+Left / Shift+Right | HEX-редактор: выделить по символу |
| Up / Down | HEX-редактор: переместить по строке |
| Shift+Up / Shift+Down | HEX-редактор: выделить по строке |
| PageUp / PageDown | HEX-редактор: страница вверх/вниз |
| Shift+PageUp / Shift+PageDown | HEX-редактор: выделить по странице |
| Home / End | HEX-редактор: начало/конец строки |
| Shift+Home / Shift+End | HEX-редактор: выделить до начала/конца строки |
| Ctrl+Home / Ctrl+End | HEX-редактор: начало/конец документа |
| Ctrl+Shift+Home / Ctrl+Shift+End | HEX-редактор: выделить до начала/конца документа |
| Ctrl+A | HEX-редактор: выделить всё |
| Ctrl+Z | HEX-редактор: отменить |
| Ctrl+Y / Ctrl+Shift+Z | HEX-редактор: повторить |
| Ctrl+X | HEX-редактор: вырезать |
| Ctrl+C | HEX-редактор: копировать |
| Ctrl+V | HEX-редактор: вставить |
| Insert | HEX-редактор: переключить вставку/замену |
| Backspace | HEX-редактор: удалить предыдущий байт |
| Delete | HEX-редактор: удалить текущий байт |
