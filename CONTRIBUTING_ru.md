# Контрибьюция

Спасибо за ваш интерес к проекту Cremniy.  
Любая помощь в улучшении проекта приветствуется.

## Способы контрибуции

Вы можете помочь несколькими способами:

- сообщать об ошибках
- предлагать новые функции
- улучшать документацию
- отправлять pull request

## Сообщение об ошибках

Если вы нашли ошибку, пожалуйста, создайте Issue и укажите:

- чёткое описание проблемы
- шаги для воспроизведения
- вашу ОС и окружение

## Предложение функций

Если у вас есть идея для новой функции, создайте Issue и опишите её.

## Pull requests

Перед отправкой pull request:

1. Сделайте fork репозитория
2. Создайте новую ветку
3. Внесите ваши изменения
4. Создайте pull request с понятным описанием

## Признание

и упомянуты в конце каждого видео на [YouTube-канале](https://www.youtube.com/@igmunv)
