## 📌 О проекте

**Cremniy** — среда разработки для низкоуровневого программирования.

Она объединяет инструменты работы с бинарными файлами,
памятью и системным кодом в одном приложении.

### Проект ориентирован на:

- разработчиков системного ПО
- reverse-engineers
- специалистов по информационной безопасности
- разработчиков embedded-систем

## ✨ Возможности

### Текущие:

- Редактор кода
- HEX-редактор

### Планируется:

- Дизассемблер
- Отладчик
- Визуализация памяти
